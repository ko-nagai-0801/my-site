### リンク

- **https://ko-nagai-0801.github.io/lunchette-demo/**

## 目的

- 小規模サイト（LP相当）を、**実運用を想定した構成**で整える
- **レスポンシブ**・**読みやすさ**・**情報の流れ**を崩さずに改善していく
- 公開後も手直ししながら、制作プロセスを作品として残す

## 工夫した点

- **情報設計**：最初に「何のサイトか」が伝わる導線を意識
- **レスポンシブ**：画面幅が変わっても読みやすい余白・並びを優先
- **セマンティックなHTML**：見た目だけでなく、意味のあるマークアップを意識
- **運用前提**：将来差し替え・更新がしやすいように、構造をシンプルに維持

## 使用技術

- HTML / CSS（スタイル設計）
- レスポンシブ対応
- BootStrap 5
- GitHub Pages（静的ホスティング）
